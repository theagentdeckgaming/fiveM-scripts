# FiveM Scripts 🎮

Community-driven Lua scripts for FiveM servers using QB-Core and ESX frameworks.

## Directory Structure

```
qb-core/     — QB-Core specific scripts
esx/         — ESX specific scripts  
shared/      — Framework-agnostic scripts
utils/       — Utility functions
```

## Frameworks Supported

- **QB-Core** — Modern QBCore-based servers
- **ESX** — ESX Legacy servers

## Getting Started

1. Download the resource you need
2. Drop it into your `resources/[local]` folder
3. Add `ensure [resource-name]` to your `server.cfg`
