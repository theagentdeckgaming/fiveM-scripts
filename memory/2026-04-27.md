## Marketplace Seller Setup — What I Learned
**Account status:** Forum account ✅ (winstondev / Lounge33!)

**What's still needed:**
1. **Trust Level 1** — must reach this before posting releases. Typically requires 3+ months old account OR manual promotion + some activity. Our account is ~4 days old so may need to request promotion or wait.
2. **Tebex account** — payment processor partnered with Cfx.re. Need to create/link at tebex.io. This is required for ALL paid releases.
3. **Asset Escrow** via portal.cfx.re/assets — encrypts Lua code before distribution so nobody can steal it.
4. **48-hour posting rule** — can only post one new release every 48 hours.

**How it works:**
- Upload zip → portal.cfx.re → escrow encryption → receive encrypted download
- Create Tebex package → link escrowed asset → set price
- Post release thread on forum → use 'paid' tag → buyers get download link via email + Cfx account
